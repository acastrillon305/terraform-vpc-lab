# Terraform VPC Lab

Sample Terraform project demonstrating how to provision a secure AWS VPC with networking and security components.

## Problem
Cloud environments need secure and repeatable infrastructure provisioning.

## Solution
This lab uses Terraform to define core AWS networking resources using Infrastructure as Code (IaC).

## Resources Included
- VPC
- Public Subnet
- Security Group
- AWS Provider Configuration

## Skills Demonstrated
- Terraform
- Infrastructure as Code
- AWS Networking
- Security Groups
- Cloud Provisioning
- DevOps

## Example Use Case
Provision a basic AWS network foundation for web applications or cloud workloads.
